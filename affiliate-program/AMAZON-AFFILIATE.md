# BECOMING AN AMAZON AFFILIATE

The process is a straight forward step by step guide in [Amazon Affiliates](https://affiliate-program.amazon.com/welcome) page.

## How to obtain your ID

At the end of the process you will obtain an ID that will be used in all the links redirecting to the products in Amazon.

## How to create the URL to be used in your product

Login to the [Amazon Affiliates](https://affiliate-program.amazon.com/welcome) dashboard and it go to the Product Linking page. You will be prompted with a search input text that you can use to find the products that you want to refer to. Once You find it, it will have a *Get Link* button that will provide the HTML where you link will be stored. Apply the legendary technique: copy and paste.
